# Contributing

Thanks for your interest in contributing.

## How to Contribute
1. Fork the repository and create your branch from `main`.
2. Make your changes with clear commit messages.
3. Run PHP lint before submitting.
4. Open a Pull Request with a clear description.

## Development Notes
- PHP 8+ and XAMPP are expected.
- Use the existing folder structure.
- Follow security practices for authentication and data handling.

## Commit Messages
Use this format:
- `feat(scope): summary`
- `fix(scope): summary`
- `chore(scope): summary`
